游戏机制——高级游戏设计技术

发表于 2018-10-22 | 分类于 notes

第一章

影响游戏结果

  1. 使游戏结果不可预测的方法：偶然性，让玩家做选择，设计复杂游戏规则
  2. 在游戏规则所能做到的事中，为玩家提供有趣的选择尤其重要
    偶然性作为辅助

游戏机制

  1. 物理，内部经济，渐进机制，战术机动，社交互动
  2. 又可分为离散和连续
  3. RPG游戏的游戏性主要来自于内部经济和渐进机制，其中内部经济用装备和经验来定制角色或队伍，渐进机制为玩家设立目标的故事情节和任务
  4. 游戏的物理机制往往是连续的，而游戏的内部经济往往是离散的，离散的机制更能够通过现实来模拟实现（早期物理规则依旧是离散的）
    物理机制与头脑策略的结合，对游戏的可玩性影响很大

游戏设计流程

  1. 概念设计：

    • 决定总体概念，目标受众，玩家扮演的角色，且一旦决定不应被改变它们
    • 可以先尝试设计实验性的基本机制，并不用考虑太多

  2. 详细设计：

    • 创建游戏关卡和具体机制，剧情，美术等
    • 让开发团队进行短期的迭代开发循环，并逐渐增加新的功能和原型
    • 找开发组外部的目标玩家进行试玩

  3. 调整阶段：

    • 进行润色，不要增加更多的特性，特性冻结，因为调整是一个减法的过程，减掉价值微小的功能

  4. 设计文档：

    • 活文档，保持更新
    • 明确说明集体目标，防止分歧
    • 记录设计过程，写下设计决定，以保证开发的后期保持正轨。

第二章

渐进和突显

  1. 渐进型游戏需要大量的数据，并提供很多预先设计好的挑战，适合用于叙事（更倾向于线性流程游戏？）
  2. 突显型游戏则只要很小的预设资料，由玩家的决定和过程中随机的发展产生千变万化的可能性
  3. 渐进型游戏往往概率空间都比较深，可控制的选项会比较少，流程长
  4. 突显型游戏则选项众多，但是流程一般不长，不然就会让玩家由于前期的小错误导致不能通关

概率空间

  1. 概率空间的面积能决定游戏的可玩性
  2. 相对简单的多个离散规则可以创造非常复杂的多变性
  3. 玩法复杂度越过某一个界的时候，概率空间会激增，游戏复杂度飞跃,但是并不是玩法复杂度越高，游戏复杂度就越高（游戏复杂度可能是指混沌程度？或者概率空间的大小？）

辐射式布局

  1. 以中央房间作为起点，引发多个子区域，并且会频繁返回中心
  2. 优点在于能让玩家自主选择任务完成顺序
  3. 能减少玩家重复路过同一区域的情况

渐进与突显相结合

  1. 经历一个突显关卡设置渐进的剧情，交叉使用
  2. 在关卡中用融入剧情，让剧情不只是作为外围框架（星际争霸2）

第三章

复杂系统

  1. 四个阶段

    • 秩序
    • 周期性系统（易于预测）
    • 突显行为（突显性系统）
    • 混沌

  2. 设计一个突显性游戏，必须让各种的元素保持一个平衡，体现出突显游戏的特点

  3. 复杂系统中最有趣的行为不是个体部分，而是个体导致的整体行为

动态行为的系统三个关键特性

  1. 系统能用相对简单的方式将每个组成部分个别地描述出来
  2. 系统必须支持远程信息传递，一个部分的改变能够跨越较远的时间和距离让其他部分也改变（蝴蝶效应）
  3. 系统组成单元的活跃程度能够有效反应出系统行为的复杂度

反馈循环

负反馈使系统趋于平衡，正反馈让系统失去稳定

突显分类

  1. 微小突显（有意突显）：很明显地进行某种反馈，且只存在于同一规模的各个元素中，其行为是可预测的
  2. 弱突显：从系统的不同级别形成从上至下的反馈
  3. 多重突显：在整个系统中有多种反馈的存在（股票：小范围正反馈，大范围负反馈）
  4. 强突显：强突显是多层级的突显，其中最高层级所产生的突显行为可与最低层级上的中介体分离开来
    ( 个人理解：从下至上让系统进行较大变化的反馈 )

第四章

资源

  1. 资源分为有形资源和无形资源：

    • 有形往往有物理的存在形式
    • 无形大部分都只是数据

  2. 也可分为抽象和具体：

    • 抽象资源并不易被玩家察觉
    • 抽象和无形并不是等价的

实体

  1. 资源的具体数目储存在实体中，如计时器
  2. 实体可分为简单实体和复合实体，简单实体只储存一个数值，复合则是多个数据，简单实体是复合实体的属性

经济系统中的四个功能

  1. 来源：
    可凭空产生新资源的并储存在某个实体的机制，可由事件触发，也可以持续运行
  2. 消耗器：
    与来源相反，将游戏中存储的资源永久消除
  3. 转换器：
    能将一种资源变成其他的形式
  4. 交易器：
    根据交换规则将某种资源从一个实体转换到另一个实体，同时将另一种资源转移回来的机制

从机制到走势图

  1. 负反馈引发经济的均衡
    均衡也可以收到其他因素的影响使均衡值改变
  2. 正反馈产生利滚利，可以让玩家快速积累经济
  3. 正负反馈可能会带来互相依赖和死锁（可引入新的资源获取方式破解死锁？）

正负反馈的具体影响

  1. 正反馈可以造成滚雪球的效应，让有优势的优势更大，同时也可以有破坏性的正反馈，加强负面效果
  2. 负反馈造成的平衡往往由其他因素影响，所以会是一个动态平衡
  3. 游戏中经常利用双方的相对位置而不是绝对位置来决定反馈力度，利用在正反馈中可以放大实力相近玩家之间的小失误，
    利用在负反馈中则是皮筋约束，让双方的相对位置总是保持在一个 动态 的稳定状态，这样落后的也有机会翻盘，但是总体优势不会大于领先的玩家

内部经济在游戏中的应用

  1. 补强物理机制

    • 动作类游戏中，给予玩家连击计分，鼓励玩家追求高分，但同时也让玩家置于危险之中
    • 增益道具（弹药等），也可以作为玩家击败某一敌人的奖励

  2. 影响游戏进程

    • 用类似于二段跳的抽象资源使游戏进程改变（也可以是具体）
    • 要防止死锁

用内部经济扩大概率空间

  1. 用来管理角色成长属性
    • 在MMORPG防止某种成长组合方式过于出众（使用负反馈）
    • 保证概率空间足够大，不要让玩家一周目就体验了游戏的所有内容
    • 最好让玩家有多种过关方式
  2. 经济构建类游戏
    • 不要把所有经济构件都一次提给玩家，让玩家在过程中解锁
    • 留意超经济结构，既要有超经济结构使游戏更有战略性，又要用其他方式防止超经济结构一家独大(使用破坏性正反馈)
